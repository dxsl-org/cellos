Expand description

A verifying TLS crypto provider for ViCell’s embedded build.

Constructed per-handshake inside socket::TlsSocketEntry::handshake(). Never stored as a global — the verifier holds per-connection transcript state.

pub fn new(rng: ViRng, hostname: &str) -> Result<Self, TlsError>

Build a provider anchored to the build-selected CA, with hostname verification.

hostname must match the server’s certificate CN or SAN. An empty hostname is rejected by handshake() before this is called, so set_hostname_verification here is always called with a non-empty value.

§Errors

Returns TlsError::InsufficientSpace if hostname exceeds 64 characters (the pki::CertVerifier limit documented in spike-00-findings.md).
