Expand description

Typed service message handler with GAT-based request lifetime.

Implement this on your service struct. Request<'de> may borrow from the receive buffer (e.g. VfsRequest<'de> borrows path strings), while Response must be fully owned (it is serialized before the buffer is reused).

§Lifetimes

The 'de lifetime in Request<'de>: Deserialize<'de> ties the decoded request to the receive buffer for that call. The buffer is not reused until handle returns, so borrowed data in the request remains valid for the entire handler call.
