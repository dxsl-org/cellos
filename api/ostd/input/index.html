Generic input event client — focus registration and event polling for any Cell. Generic input event client for any Cell.

Provides focus registration and non-blocking event polling without any dependency on libs/viui. ViUI apps should use viui::input_bridge which wraps this module and converts events to viui::Event.

§Usage

use ostd::input::{request_focus, poll_events, InputEvent};

// Once at startup:
while !request_focus() { ostd::task::yield_now(); }

// Every tick:
for ev in poll_events(32) {
    if let InputEvent::Key(k) = ev { /* handle key */ }
}
