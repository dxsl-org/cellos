DMA buffer helpers for Tier-1 Driver Cells (contiguous allocation + IOMMU authorisation). DMA buffer helpers for Tier-1 Driver Cells.

Driver Cells allocate physically contiguous memory for DMA rings via sys_grant_alloc (which uses allocate_contiguous internally), then authorise the IOMMU via sys_grant_dma. This module wraps both into a safe interface.

§Usage

ⓘ
let buf = DmaBuf::alloc(4).expect("OOM");      // 4 pages (16 KiB)
buf.authorize(0x0300).expect("IOMMU deny");     // BDF 0:3:0
let phys = buf.phys();                         // program into NVMe/e1000 regs
