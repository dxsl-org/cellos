Struct DmaBuf

pub struct DmaBuf { /* private fields */ }
Expand description

A physically-contiguous, page-aligned DMA buffer backed by a Grant region.

In SAS the virtual address equals the physical address (identity mapping), so phys and virt return the same value.

Drop does NOT free the buffer — call DmaBuf::free explicitly. Driver Cells typically hold DMA buffers for their entire lifetime.

pub fn authorize(&self, bdf: u32) -> Result<u64, SyscallError>

Authorise DMA for the PCIe device at bdf covering this buffer.

Must be called before issuing any DMA operation. The IOMMU maps [phys, phys + size) for this BDF in the calling cell’s domain. Returns Ok(iova) on success (iova == phys in SAS).
