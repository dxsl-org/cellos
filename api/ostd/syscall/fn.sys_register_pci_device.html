Register a discovered PCI device with class and BAR0 info.

Populates the kernel PCI_DEVICES list so sys_find_pcie_device queries work without a kernel-side ECAM scan. Call once per discovered device after scan_and_register. Requires singleton PlatformCap (allowlist bit 53).

  • bdf — (bus << 16) | (dev << 8) | fun
  • cls — (class << 16) | (subclass << 8) | prog_if
  • bar0_base — physical base of BAR0, or 0 if absent
  • bar0_size — probed BAR0 size in bytes, or 0
