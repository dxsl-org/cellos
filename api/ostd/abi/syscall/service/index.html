Module service 

Expand description

Well-known service IDs for the kernel Service Registry (RegisterService / LookupService). Stable ABI — values must not change once shipped. 0 is reserved to mean “no provider” (the value LookupService returns when nothing is registered).

A client resolves the live provider tid of a service by LookupService(service::VFS) instead of hard-coding a tid, so it transparently reconnects when the supervisor respawns that service under a new tid.
