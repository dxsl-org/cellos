Provided Methods§

fn stat(&self, path: &str) -> Result<Stat, ViError>

Return metadata for path. Returns Err(NotFound) if absent.

fn rmdir(&self, path: &str) -> Result<(), ViError>

fn readdir(&self, path: &str) -> Result<Vec<DirEntry>, ViError>

List directory entries at path.

Returns Err(NotADirectory) if path exists but is a file. Returns Err(NotFound) if path does not exist. Returns Err(NotSupported) if the filesystem does not implement directory listing.
