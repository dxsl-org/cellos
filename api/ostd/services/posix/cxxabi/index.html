Module cxxabi 

Expand description

Minimal C++ ABI stubs for freestanding C++ (no exceptions, no RTTI).

SAFETY: These stubs ONLY support -fno-exceptions -fno-rtti compiled C++ code. Do NOT attempt to use exceptions, RTTI, or STL containers — they require full libcxxabi/libstdc++ which is SAS-unsafe. Use Tier 3b Linux VM instead.

Signals that function-local static initialization failed, leaving the guard clear so the next execution retries initialization.
__cxa_guard_acquire⚠
Thread-safe static local init guard — single-threaded stub. C++ emits __cxa_guard_acquire/release around function-local statics. In ViCell’s single-threaded cells, a simple flag suffices.
__cxa_guard_release⚠
Marks the function-local static guarded by guard as initialized.
__cxa_pure_virtual⚠
Called when a pure virtual function is invoked (programming error). Terminates the cell immediately — equivalent to a Rust panic.
abort⚠
abort() — terminates cell immediately. No cleanup, no atexit handlers. This is the correct behavior for SAS: kernel reclaims all resources.
