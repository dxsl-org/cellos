Module posix 

Expand description

POSIX shim layer — Tier 1b C library support for ViCell cells.

Provides malloc, string ops, file I/O, network sockets, entropy, math, stdio, and setjmp as #[no_mangle] C-ABI symbols. No picolibc or toolchain libm required — math is backed by the Rust libm crate.

Activates automatically on supported arches; cells must NOT link -lm.
