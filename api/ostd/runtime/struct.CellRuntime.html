Struct CellRuntime

pub struct CellRuntime { /* private fields */ }
Expand description

Builder that arms the watchdog heartbeat and runs the cell event loop.

§Usage

CellRuntime::new()
    .heartbeat(500)             // 5-second watchdog (500 × 10 ms ticks)
    .help("Usage: foo <file>")  // auto-handle -h / --help
    .run(my_handler);

pub const fn new() -> Self

Create a builder with a 5-second heartbeat default (500 ticks × 10 ms).

Source

pub const fn heartbeat(self, ticks: u64) -> Self

Override the watchdog interval. ticks × 10 ms before the kernel kills a hung cell. Pass 0 to disable the heartbeat.

Source

pub const fn no_heartbeat(self) -> Self

Disable the watchdog heartbeat for this cell.

Source

pub const fn help(self, usage: &'static str) -> Self

Register a usage string for automatic -h / --help handling.

If the cell is spawned with -h or --help in its args, usage is printed to the console and the cell exits before entering the event loop. Convenience alternative to calling [ostd::args::check_help] manually in the AppEvent::Init arm.

Source

pub fn run(self, handler: impl FnMut(&mut AppContext, AppEvent)) -> !

Arm the heartbeat, check for --help, fire AppEvent::Init once, then run the event loop.

Never returns.
