Struct GlyphAtlas

pub struct GlyphAtlas { /* private fields */ }
Expand description

Scalable glyph rasterizer wrapping a fontdue::Font.

Rasterized bitmaps are cached by (codepoint, size_bits) — repeated draws for the same character at the same size are essentially free.

§Usage

let atlas = GlyphAtlas::new(include_bytes!("../assets/font.ttf")).unwrap();
let (metrics, bitmap) = atlas.rasterize('A', 16.0);

pub fn rasterize(&mut self, c: char, px: f32) -> (GlyphMetrics, &[u8])

Rasterize c at px pixels tall.

Returns (GlyphMetrics, coverage_bitmap). The bitmap is 1 byte per pixel stored row-major, top-left origin, width × height bytes total. Repeated calls for the same (c, px) return the cached result.
