Macro declare_cluster 

macro_rules! declare_cluster {
    (mode = $mode:ident, name = $name:literal) => { ... };
}
Expand description

Emit the __ViCell_cluster ELF section declaring this Cell’s cluster mode.

The section is read by the kernel loader at spawn time. Cells that omit this macro default to ClusterMode::Isolated and are unaffected by cluster operations.

§Example

ⓘ
api::declare_cluster!(mode = Private, name = "robots");

mode must be one of Isolated, Public, or Private. name is the cluster name string used to derive the ClusterId.
