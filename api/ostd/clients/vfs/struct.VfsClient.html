Ergonomic client for the VFS service.

Wraps VfsRef and hides request construction + postcard encoding. Each method allocates a 4 KiB response buffer on the stack for the duration of the call (freed on return).

§Large files

read_file uses GetFile, which transfers the entire file contents in one IPC message. Files larger than ~4 KB should use the Grant-based API (VfsRequest::ReadGrant) directly via VfsRef.
