A TCP connection handle implementing [embedded_io::Read] and [embedded_io::Write].

Obtained via TcpStream::connect. The underlying socket is closed when TcpStream is dropped (RAII — Law 8).

§Blocking contract

[Read::read] blocks (with cooperative yields) until at least one byte is available, matching the [embedded_io] blocking contract. The net service returns data non-blockingly per call; the loop in read handles the wait.

§Write chunking

[Write::write] sends at most [MAX_TCP_WRITE] bytes per call due to the IPC buffer limit. Use [write_all][embedded_io::WriteExt::write_all] when all bytes must be delivered.
