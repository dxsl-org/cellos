pub fn tcp_send(&mut self, id: SocketId, data: &[u8]) -> ViResult<()>

Send data over an established TCP socket.

The net service replies Data(n) where n (4-byte LE) is the number of bytes the socket accepted — 0 while the connection is still completing its handshake (SynSent). Returns WouldBlock if not all bytes were accepted so the caller can retry (after a yield) once the socket is Established. data should be ≤ the socket’s tx buffer (4 KiB) per call.
