x86_32 (IA-32) Hardware Abstraction Layer.

Targets i686-unknown-none (custom JSON, bare-metal, multiboot1 via QEMU -kernel). Nano profile: GDT + IDT init, COM1 UART, cooperative context switch. Sub-modules gated on #[cfg(target_arch = "x86")] so this file compiles on all hosts without emitting 32-bit-only inline asm.
