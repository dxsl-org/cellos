BCM2837 GPIO Driver Cell — Raspberry Pi 3 (BCM2837).

Base: 0x3F200000, 54 pins, 32-bit GPFSEL/GPSET/GPCLR/GPLEV/GPREN/GPFEN. Pin function select: 3 bits per pin, 10 pins per GPFSEL register.

IRQ routing: BCM2835 legacy controller bank0 (pins 0–27) / bank1 (pins 28–45). The kernel IRQ handler notifies the MMIO owner via opcode 0xA0 when a GPIO edge fires — same protocol as the PL061 driver.

Pull-up/down: BCM2837 requires a 3-step GPPUD + GPPUDCLK sequence (≥150 cycles hold time). This driver omits the timing (no busy-wait in no_std), so pull configuration is caller responsibility if needed.
