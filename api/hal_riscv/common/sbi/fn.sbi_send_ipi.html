pub fn sbi_send_ipi(
    hart_mask: usize,
    hart_mask_base: usize,
) -> Result<(), usize>
Expand description

Send a software IPI to all harts selected by hart_mask.

Bit N in hart_mask targets hart hart_mask_base + N. Pass hart_mask_base = 0 for absolute hart IDs (e.g. 1 << 1 to ping hart 1). The IPI sets SSIP on each target; the target’s trap handler sees scause=1.
