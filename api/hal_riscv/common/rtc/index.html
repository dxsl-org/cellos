Goldfish RTC driver for RISC-V (QEMU virt machine).

Init: platform::init() calls init(base) with the MMIO base from DTB node google,goldfish-rtc. MMIO layout (QEMU hw/rtc/goldfish_rtc.c): offset 0x00: TIME_LOW — reading latches TIME_HIGH; returns bits 31:0 of epoch_ns offset 0x04: TIME_HIGH — returns latched bits 63:32 of epoch_ns
