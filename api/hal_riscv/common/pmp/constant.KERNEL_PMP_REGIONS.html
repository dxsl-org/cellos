Intended kernel protection regions for entries 2–6.

Entries 0–1 are owned by OpenSBI. These descriptors should be written to PMP entries 2–6 by the M-mode firmware shim before entering S-mode.

Physical addresses are for the QEMU virt machine with ViCell’s default linker layout (ORIGIN = 0x80200000). Adjust if the layout changes.
