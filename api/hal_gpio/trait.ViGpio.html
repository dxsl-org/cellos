Expand description

GPIO controller trait for Driver Cells.

Implemented by cells/drivers/gpio-pl061 (and future per-chip impls). #![forbid(unsafe_code)] Cells use this via the safe MmioRegion abstraction.

§Invariants

  • Pin indices are 0-based; out-of-range → Err(ViError::InvalidInput).
  • write_pin on an input pin → Err(ViError::InvalidInput).
