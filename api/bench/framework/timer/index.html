High-resolution timer for benchmark measurements.

Reads the kernel’s monotonic tick counter via sys_get_time. On RV64 QEMU this corresponds to mtime at ~10 MHz. Convert ticks → ns by dividing by ticks_per_ns() (queries the Config Cell; falls back to 10 MHz assumed).

Constants§

FALLBACK_FREQ_HZ 🔒
Timer frequency assumed when the Config Cell is unavailable (10 MHz).
NS_PER_TICK
Nanoseconds per tick — derived from FALLBACK_FREQ_HZ. Multiply tick deltas by this to get nanoseconds.
