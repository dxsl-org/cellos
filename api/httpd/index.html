httpd — minimal HTTP/1.0 file server for ViCell.

Usage: httpd <vfs_path>

Listens for TCP connections on . For each connection, reads the HTTP request (discards it), reads <vfs_path> fresh from VFS on every request (no caching), and responds with HTTP/1.0 200 OK + current file content. Returns HTTP 404 when the file is absent or empty. Loops forever, serving one connection at a time.
