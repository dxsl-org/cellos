Expand description

Software cursor sprite — compile-time 16×16 BGRA8888 arrow bitmap.

The arrow points top-left. Each row is encoded as a u16 bitmask (MSB = leftmost pixel). Bit set = white fill pixel; neighbouring unset bits that are outline candidates = black outline pixels.

Black outline pixel — slightly transparent for a softer edge (BGRA8888).
CURSOR_H

Cursor sprite width and height in pixels.
OUTLINE_MASK 🔒
Black outline mask — pixels that border the arrow fill. Derived from the fill mask: pixels adjacent to a set fill bit, but not themselves set. Pre-computed to avoid per-pixel expansion at runtime.
WHITE 🔒
White cursor fill pixel (BGRA8888).
