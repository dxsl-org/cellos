Compositor Service Cell.

Manages a z-ordered set of cell surfaces, blends them into a screen framebuffer, and flushes dirty regions to the VirtIO GPU via the GpuFlush kernel syscall.

Input routing: on startup the compositor registers as the input service’s focus endpoint. All InputEvent IPC frames are dispatched to input_handler, which forwards keyboard events to the focused surface owner and updates the cursor position on mouse move.
