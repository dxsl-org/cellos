fn blit_surface(&mut self, s: &SurfaceState)

Blit one surface’s pixels into the screen FB at its screen position.

Clips to the screen boundary; surfaces that are partially off-screen are rendered up to the edge.

Source

fn composite_cursor(&mut self, cx: i32, cy: i32, dirty: Rect)

Alpha-blend the 16×16 cursor sprite at (cx, cy) into the FB.

Only pixels that fall inside dirty are written — avoids painting outside the region that will be flushed to the GPU this frame. Clips the sprite to screen bounds so a cursor near an edge is safe.

Pixel format is BGRA8888 (native VirtIO GPU order). Alpha blend formula: out = src * alpha/255 + dst * (255 - alpha)/255

Source

fn flush_rect(&mut self, dirty: Rect)

Flush dirty_rect from the screen FB to the GPU.

Copies the dirty region into the pre-allocated staging buffer (no heap allocation) then hands the sub-slice to the kernel. Clamps to screen boundary.
