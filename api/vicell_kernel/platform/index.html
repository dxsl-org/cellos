Module platform 

Source
Expand description

Platform peripheral discovery via DTB.

Call platform::init(dtb_ptr) at kmain before any driver init. All drivers read MMIO addresses via platform::with(|p| p.xxx). Falls back to QEMU virt defaults when the DTB is absent or a compatible node is not found.

Call ordering invariant: init → uart::init → hal::ARCH.init (which calls plic::init internally). Any call to with before init panics.
