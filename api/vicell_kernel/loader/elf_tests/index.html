Module elf_tests 

Source
Expand description

Boot-time tests for the ELF loader and relocation engine.

Functions are pub so kernel/src/loader.rs can invoke them from the boot-time test runner. Every function logs with log::info! and asserts with standard assert!/assert_eq! — a failing assert panics the kernel, which is intentional (hard failure = detected early).
