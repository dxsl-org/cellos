Expand description

Fleet root Ed25519 public key (trust anchor; lives in the kernel TCB, not in mutable VIFS1 data). dev-policy-key feature → the dev key (so a dev-signed /POLICY.BIN verifies); otherwise a placeholder the production provisioning replaces. A zero/placeholder key fails every verify → any present policy is Invalid (fail-closed), the safe direction; absent policy still boots (dev-permissive).
