fn validate_user_buf(
    ptr: usize,
    len: usize,
    max: usize,
) -> Result<(), SyscallError>
Expand description

Validate a user-supplied (ptr, len) buffer descriptor.

Rejects: NULL pointer, zero-length when expected non-empty, lengths above max, and pointer+length arithmetic overflow.

Does NOT walk the page table to confirm the U-bit. The trap handler enables SUM only for the duration of handle_syscall, so a kernel-space ptr supplied by user code will fault on access — but the fault is far more graceful when we reject obvious garbage up front.
