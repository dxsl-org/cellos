vicell_kernel::task::hart_local

Module ready 

Source
Expand description

Per-hart ready-queue helpers for the Phase 03 work-stealing scheduler.

Lock order: SCHEDULER (global, coarse) → per-hart ready lock (leaf). steal_from_busiest holds only leaf locks — never SCHEDULER.

RT tasks (priority ≥ RealTime) are never stolen; Phase 04 will hart-pin them.
