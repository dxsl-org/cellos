Struct VirtioHal 

Source 
pub struct VirtioHal;
Expand description

VirtIO HAL adapter for ViCell.

Assumes identity mapping (VAddr == PAddr) throughout, which holds for the kernel’s current single-address-space model. Revisit if an HHDM or IOMMU is introduced: every paddr = ptr as usize line must become paddr = ptr as usize - HHDM_OFFSET.
