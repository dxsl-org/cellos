Module input_irq_ack 

Source
Expand description

Minimal VirtIO input IRQ ACK shim.

Keeps InterruptStatus cleared when the input Cell is not yet running or has crashed — without this, an unacknowledged input IRQ becomes an interrupt storm.

This is the ONLY kernel-side remnant of the old virtio_input driver. All event routing is handled by the input service Cell (cells/services/input). Full deletion of this shim is a G2 task once input Cell reliability is confirmed.
