Expand description

SDHCI host controller — PIO polling mode, no DMA, no interrupts.

The base field is the kernel-mapped virtual address of the SDHCI register block. All MMIO accesses are read_volatile / write_volatile to prevent optimisation.

Fields§

§base: usize§is_sdhc: bool

SDHC = true (block-addressed); SDSC = false (byte-addressed).

§spec_ver: u8

SDHCI spec version read from HOST_VERSION[2:0]; affects clock divider encoding.
