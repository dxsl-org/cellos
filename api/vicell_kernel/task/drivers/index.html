Module drivers

Kernel-side driver shims, capability registry, and routing tables.

Only kernel-resident drivers that satisfy the Boundary Law remain here (early-boot console, VirtIO block + MMC as G2-pending fallbacks, IOMMU, IRQ wait tables). All other device drivers live in cells/drivers/.
