pub fn register_block_driver(tid: usize)
Expand description

Record tid as the active block driver. Overwrites any previous registration.

Logged at warn! for the same reason as set_input_cell: Driver Cells are spawned by init AFTER the kernel drops its log level to Warn, this is a one-time boot-integrity event (the kernel now routes all sector I/O to this TID), and it is the marker the x86 nvme/nic integration tests assert on.
