What resource a capability refers to.

Variants§

§

File

An open file backed by the kernel-internal FS (VIFS1 / ramFS).

The file field is None while a ReadCap syscall is in progress — the kernel takes the Box out, releases the cap-table lock, performs I/O, then re-acquires the lock and puts the Box back. This prevents the global cap-table lock from being held across potentially-slow I/O. A concurrent ReadCap on the same cap while it is parked returns Err(ViError::TryAgain).
