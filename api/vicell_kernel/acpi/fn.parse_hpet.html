Parse HPET table for event timer block MMIO address.

HPET table layout (after 36-byte SDT header): offset 36: event_timer_block_id (u32) — hardware rev, comparators, etc. offset 40: base_address (GAS, 12 bytes) GAS[0]: address_space_id (u8) — 0 = MMIO GAS[4..12]: address (u64) offset 52: hpet_number (u8) offset 53: minimum_tick (u16) offset 55: page_protection (u8)
