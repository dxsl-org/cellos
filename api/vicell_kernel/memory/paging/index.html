Paging management for ViCell kernel.

Architecture dispatch: riscv64, aarch64 — SV39 / AArch64 page tables via HAL PageTable x86_64 — 4-level PML4, dedicated path (own walker + HHDM-aware) riscv32, x86, arm — bare physical (no page tables); stubs return NotSupported
