pub struct GossipTable {
    claims: [Option<ClaimRecord>; 32],
    epoch_counter: u64,
    my_machine_id: u64,
}
Expand description

Per-broker view of all active task claims in the cluster.

A claim is “active” until: (a) a TaskRelease with epoch ≥ claim.epoch is received from the same machine for the same task_id, OR (b) the sender’s peer entry times out (peer loss → all its claims evicted).
