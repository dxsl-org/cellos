STUN Binding Request client (RFC 8489, §5.3).

Discovers the machine’s reflexive (public) IPv4:port behind NAT. UDP socket is opened, queried, then closed — zero persistent socket budget.

§UdpRecv header

The net cell prepends a 6-byte src-addr header to every UDP receive buffer: [0..4] source IPv4 (BE) [4..6] source port (BE) STUN parsing skips these 6 bytes before reading the STUN frame.
