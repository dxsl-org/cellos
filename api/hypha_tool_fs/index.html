Hypha tool-fs — filesystem tool cell (P2).

Receives [AgentToolRequest::Invoke] from core, dispatches to [ostd::clients::VfsClient] (VFS service via IPC — no block_io cap needed), and replies [AgentToolResponse::Ok] or [AgentToolResponse::Err].

Supported tools:

  • read_file {"path":"..."} → {"content":"..."}
  • write_file {"path":"...","content":"..."} → {"ok":true}
  • list_dir {"path":"..."} → {"files":["a","b",...]}
