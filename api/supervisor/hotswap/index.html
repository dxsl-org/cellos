Module hotswap 

Source
Expand description

Hotswap orchestration ported from kernel/src/cell/hotswap.rs.

Kernel mechanisms replaced by SupervisorCap syscalls: set_task_frozen(tid) → sys_freeze_cell(tid) unfreeze_task(tid) → sys_resume_cell(tid) exit_task_internal(..) → sys_kill_cell(tid, 0xAAAA_AAAA)

Polling replaced by yield-and-retry loops using sys_yield() + sys_state_restore() and sys_query_hotswap_ready(). The timeouts are calibrated to scheduler tick rate (one tick ≈ 10 ms) — HOTSWAP_MAX_ITERS×YIELD_COST_TICKS ≈ 5 seconds.
