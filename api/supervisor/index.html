Crate supervisor 

Source
Expand description

Supervisor Cell — Tier-1 Trusted Cell for hotswap/snapshot orchestration.

Holds SupervisorCap (granted by the kernel loader at spawn — path-based grant since all 8 manifest flag bits are occupied in v1) and SpawnCap (declared in manifest so init sets the bit via intersection).

Service registration (service::SUPERVISOR) is handled by init after spawning; the Supervisor Cell does not need to self-register.

On crash: init restarts the Supervisor Cell (never-die). Frozen target cells survive the restart because sys_freeze_cell state persists in the kernel.
