pub struct HotswapRequest<'a> {
    pub target_service: &'a [u8; 64],
    pub new_elf: &'a [u8; 128],
}
Expand description

Hotswap request from any authorized caller.

target_service is the null-terminated ASCII name that maps to a service ID. new_elf is the VFS path of the replacement ELF (null-terminated).
