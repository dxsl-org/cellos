1//! TLS 1.3 support for the net service cell (embedded-tls 0.19).
2//!
3//! Build flavors (mutually exclusive cargo features):
4//!   tls-roots-embedded — single pinned CA via pki::CertVerifier (G1 default)
5//!   tls-roots-full     — rustls-webpki multi-root (G2, deferred P04)
6//!   tls-insecure       — UnsecureProvider, no verification (dev/lab only)
7//!
8//! Module layout:
9//!   clock    — ViTlsClock (TlsClock impl over sys_get_wall_secs + clamp)
10//!   roots    — ca_cert() single trust anchor, cfg-selected by tls-ca-* feature
11//!   provider — ViTlsProvider (CryptoProvider with infallible verifier())
12//!   rng      — ViRng (VirtIO-RNG-backed ChaCha20)
13//!   transport — SmoltcpTlsTransport (embedded-io Read+Write over smoltcp TCP)
14//!   socket   — TlsSocketEntry (per-connection TLS state + handshake)
15//!   block_on — blocking executor shim
16
