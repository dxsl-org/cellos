ARM AArch32 (ARMv7-A) Hardware Abstraction Layer.

Targets armv7a-none-eabi bare-metal. Sub-modules gated on #[cfg(target_arch = "arm")] to avoid AArch32 inline asm bleeding into non-ARM builds (e.g. AArch64 host compilations).
