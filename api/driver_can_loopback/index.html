In-memory CAN loopback driver — no MMIO, no capability required.

send_frame pushes into a fixed-size ring buffer. recv_frame pops from the same buffer.

This validates the ViCan trait interface and frame encoding on QEMU without any hardware or resource registry changes.

Capacity: 32 frames. Overflow drops the oldest frame and sets overflow_count.
