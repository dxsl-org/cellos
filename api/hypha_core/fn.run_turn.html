Expand description

Run one agentic turn: call the LLM, dispatch any tool requests, and return the final text reply. Loops up to MAX_TOOL_ROUNDS times to handle chained tool calls. Tool interactions are appended to working_prompt in-place (not stored in the permanent conversation — only the final reply is kept).
