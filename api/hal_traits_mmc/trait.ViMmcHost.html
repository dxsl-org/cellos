Expand description

SDHCI host-controller interface (PIO polling mode).

Implementors: SdhciController (kernel). All methods are synchronous. Async DMA support is deferred to a future phase.

§Invariants

  • read_block / write_block operate on exactly buf.len() bytes. Callers must ensure buf.len() equals the block size set on the controller.
  • send_cmd must not be called while CMD_INHIBIT or DAT_INHIBIT is asserted; callers are responsible for polling readiness first.
