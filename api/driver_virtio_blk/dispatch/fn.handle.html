pub fn handle(dev: &mut BlkDevice, data: &[u8], out: &mut [u8; 513]) -> usize
Expand description

Handle one incoming IPC message. Writes the reply into out and returns the number of bytes to send back.

Read OK: [0x00] ++ sector_data, returns 513. Write OK: [0x00], returns 1. Error: [0x01], returns 1.
