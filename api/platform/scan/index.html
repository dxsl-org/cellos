Module scan 

Source
Expand description

ECAM bus-0 scanner for the Platform Cell.

Walks all 32 device slots on bus 0 and registers each discovered MMIO BAR with the kernel via sys_register_pcie_bar. This lets Driver Cells subsequently claim individual BARs through sys_request_mmio.

BAR size probing follows PCI 3.0 §6.2.5.1: write 0xFFFFFFFF, read back, mask lower 4 bits, compute ~mask + 1. Memory decode is disabled in the command register before each probe and restored after to prevent the device from responding to MMIO accesses during the transient probe window.
