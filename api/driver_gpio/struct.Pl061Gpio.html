Acquire exclusive GPIO MMIO access from the kernel.

Source

pub fn read_mis(&self) -> ViResult<u8>

Return the masked interrupt status register (GPIOMIS).

Each set bit indicates a pin whose edge IRQ fired AND was enabled. Call this after receiving a GPIO_IRQ_NOTIFY (opcode 0xA0) from the kernel to discover which pins triggered, then call clear_irq(mask) to ACK them.

Source

pub fn clear_irq(&mut self, mask: u8) -> ViResult<()>

Acknowledge (clear) GPIO interrupts for the pins in mask.

Writes GPIOIC without touching GPIOIE, so the interrupt remains enabled for the next edge. Call after reading GPIOMIS and processing all set bits.
