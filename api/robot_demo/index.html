Reference robot demo — G1 graduation criterion 8.

Pipeline: SHT3x I2C sensor read → threshold compute → GPIO relay actuator → MQTT telemetry. Falls back to synthetic data when no I2C slave responds (expected on QEMU). Falls back to pure simulation when GPIO is unavailable (RISC-V target).
