Crate hypervisor 

Source
Expand description

Hypervisor Service Cell — boots an ARM64 Linux guest via EL2 Stage-2 MMU.

Reads vmlinuz + initrd.gz from VIFS1, builds a minimal DTB, maps 128 MiB of guest RAM, loads all images, and runs the VmExit dispatch loop.
