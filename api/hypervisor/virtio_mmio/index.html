virtio-mmio Version=2 register-block emulator for one device slot.

Handshake: ACK(1)→DRIVER(2)→feature exchange→FEATURES_OK(8)→queue setup→DRIVER_OK(4). VERSION_1 (bit 32 = DriverFeatures high-word bit 0) is mandatory; rejected otherwise.
