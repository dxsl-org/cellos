Module vmm 

Source
Expand description

Low-level VMM syscall wrappers (220-226) for the hypervisor service cell.

These are ARM64-only at runtime (guarded by cpu_features::has_el2 at cell start). RISC-V stubs return a NotSupported sentinel so the code compiles on all targets.
