pub fn push_rx_frame( &mut self, frame: &[u8], vm_id: usize, vcpu_id: usize, net_vmio: &VirtioMmio, )

Inject one received Ethernet frame into the guest RX virtqueue.

Prepends a 10-byte zero virtio_net_hdr, fills one avail descriptor chain, advances the used ring, and injects SPI 18 to notify the guest. Silently drops the frame if the avail ring is empty (no guest buffers ready).
