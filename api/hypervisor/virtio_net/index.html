Expand description

virtio-net device model (DeviceID=1, virtio-mmio slot 2 → SPI 18).

TX: guest writes TX avail ring → VMM reads chain → L2Send to Net Cell → kernel NIC TX. RX: Net Cell L2Recv delivers frame → VMM fills RX avail descriptor → guest SPI 18.

The 10-byte virtio_net_hdr is prepended on RX and stripped on TX.
