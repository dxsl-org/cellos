fb-console — mirrors kernel user-log output to the HDMI display.

Creates a full-screen background surface in the compositor and renders incoming log bytes as white-on-black 8×8 bitmap text. Wraps at the right edge; scrolls one line up when the bottom row is reached. No UART I/O.

Requires ReadLog capability (allowlist bit 54).
