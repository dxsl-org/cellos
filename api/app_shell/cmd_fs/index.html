Module cmd_fs 

Source
Expand description

Filesystem-oriented shell built-ins: wc, head, tail, grep, mkdir, rm, rmdir, touch.

VFS-write operations (mkdir, rm, rmdir) send IPC to the VFS service cell resolved via sys_lookup_service. Read operations use the kernel’s sys_open/sys_read path.
