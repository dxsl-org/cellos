Module executor 

Source
Expand description

Shell AST executor — runs parsed commands, handles pipes and redirects.

Pipes between built-in commands are implemented via an in-memory OutputSink: each pipeline stage’s output is captured into a Vec<u8>, then passed as stdin to the next stage. SinkGuard (RAII, Law 8) ensures the sink is restored on every exit path, including early returns.
