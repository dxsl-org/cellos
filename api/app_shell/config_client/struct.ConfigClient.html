Struct ConfigClient

pub struct ConfigClient {
    resp_buf: UnsafeCell<[u8; 4096]>,
}
Expand description

Client for the Config service.

Uses typed postcard IPC (ConfigRequest / ConfigResponse) matching the config service v0.3 protocol. Resolves the live Config endpoint via the Service Registry on each call, so it transparently reconnects when the supervisor respawns Config.

§Safety invariant

resp_buf is accessed exclusively through &self via UnsafeCell. ConfigClient is only used from the shell’s single thread. Sync is manually implemented for that reason; do not share across threads.
