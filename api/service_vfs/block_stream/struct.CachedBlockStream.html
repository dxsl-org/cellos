    inner: BlockStream,

}
Expand description

fatfs block stream with an LRU sector cache.

Replaces BlockStream as the fatfs I/O backend. All sector reads are served from PageCache on hit; misses fall through to BlockStream. Writes use write-through policy (flush on every write) while backed by FAT32.
