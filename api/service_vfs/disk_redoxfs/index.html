Module disk_redoxfs 

Source
Expand description

VirtIO block device adapter for RedoxFS — maps 4 KiB RedoxFS blocks to 512-byte VirtIO sectors on MBR partition P5 (PART_SRV_BASE_LBA).

BLOCK_SIZE (redoxfs) = 4096 bytes = 8 × 512-byte sectors. block N on the RedoxFS volume → sectors PART_SRV_BASE_LBA + N*8 on disk.
