Expand description

Access rule for a single path prefix.

Fields§

§prefix: &'static str

The path prefix this rule applies to (e.g., /data/, /bin/).

§allow_read_all: bool

True if any cell may read from this prefix.

§allow_write_all: bool

True if any cell may write to this prefix. False means only cells in write_allowlist (or nobody if the list is empty).
