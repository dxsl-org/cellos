fn read_async( self: Box<Self>, buf_ptr: usize, buf_len: usize, ) -> BoxFuture<'static, FileResult<usize>>

Async Read: Takes ownership of the file handle and returns a Future.

Implementations MUST return the file handle back in the result tuple.

The buffer is passed as raw pointer/len because the Future is ’static and we cannot easily bind the lifetime of a user-space slice to it safely without complex logic. The caller (Kernel) ensures safety.

fn size(&mut self) -> ViResult<u64>

Return the file’s current size in bytes without changing the cursor position.

The default saves the current position, seeks to EOF, then restores. Override in stateless implementations (e.g. FatFile) to avoid the extra seek. Returns ViError::IsADirectory for directories.

Source

fn truncate(&mut self, _len: u64) -> ViResult<()>

Truncate the file to exactly len bytes.

Returns ViError::InvalidArgument if len > current_size; use write to extend. Returns ViError::NotSupported if the backend does not implement truncation.

Source

fn sync(&mut self) -> ViResult<()>

Flush all dirty pages to the underlying block device (fsync).

No-op on write-through implementations; wires into device flush on NVMe (G2).
