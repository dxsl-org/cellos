Expand description

Attach an app-owned Grant buffer to a compositor surface.

The app must have already called sys_grant_share(reg_id, comp_tid, 0 /* ReadOnly */). Compositor replies with [0x01] on success, [0x00] on failure. Total wire size: 24 bytes.
