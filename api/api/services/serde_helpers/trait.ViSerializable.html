Trait ViSerializable

pub trait ViSerializable: Sized {
    // Required methods
    fn serialized_size(&self) -> usize;
    fn serialize_into(&self, buffer: &mut [u8]) -> ViResult<usize>;
    fn deserialize_from(buffer: &[u8]) -> ViResult<Self>;
}
Expand description

This trait simplifies StateTransfer implementation by providing automatic serialization for common types.

Dyn Compatibility§

This trait is not dyn compatible.

In older versions of Rust, dyn compatibility was called "object safety", so this trait is not object safe.
