Struct FontContext 

Source 
pub struct FontContext {
    pub atlas: Option<GlyphAtlas>,
    pub size_px: f32,
}
Expand description

Scalable font state shared across a full frame’s paint pass.

Passed through RenderCtx to every ViNode::paint() call. Widgets that do not need scalable text ignore the atlas field entirely.

Fields§

§atlas: Option<GlyphAtlas>

Cached glyph rasterizer. None = bitmap 8×8 fallback.

§size_px: f32

Default font size in pixels used by text widgets.
