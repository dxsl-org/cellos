viui::app_runner

Struct ViApp

pub struct ViApp { /* private fields */ }
Expand description

Tick-based app runner.

Call tick() or tick_with_dt() from the cell’s main loop, passing any buffered input events and — for animation — the milliseconds elapsed since the last call.

pub fn new(root: Box<dyn ViNode>, renderer: Box<dyn ViRenderer>) -> Self

Create a new app with no scalable font (8×8 bitmap fallback).

The first tick always renders a full frame.

Source

pub fn action_queue(&self) -> OverlayActionQueue

Return a clone of the action queue.

Pass this clone to Dialog / DropDown constructors so they can push OverlayAction items without holding a borrow on ViApp.

Source

pub fn push_overlay(&mut self, widget: Box<dyn ViNode>, blocking: bool)

Push a widget as a blocking or non-blocking overlay.

blocking = true — modal: all input is consumed by the overlay, the root does not see events while it is on screen (e.g. Dialog). blocking = false — non-modal: input passes through when not consumed by the overlay (e.g. DropDown popup).

Source

pub fn pop_overlay(&mut self)

Pop the topmost overlay. No-op when the overlay stack is empty.

Source

pub fn show_toast(&mut self, config: ToastConfig)

Show a transient toast notification.

Toasts are rendered above all content and auto-dismiss after config.duration_ms milliseconds (0 = never auto-dismiss).

Source

pub fn with_font(self, font_bytes: &[u8], size_px: f32) -> Self

Override the default 8×8 bitmap font with a scalable TTF font.

font_bytes: raw TTF/OTF data (use include_bytes! for embedded fonts). size_px: default text height in pixels. Returns self unchanged when the font bytes are invalid.

Source

pub fn set_theme<T: ViTheme + 'static>(&mut self, theme: T)

Switch the active theme. The next frame will render with the new tokens.

Source

pub fn add_animation(&mut self, anim: Box<dyn Animatable>)

Register an animation that is advanced every tick_with_dt() call.

Animations typically wrap an AnimatedSignal<f32> and drive a widget signal — no widget code changes are required.

Source

pub fn tick(&mut self, events: &[Event]) -> bool

Process events and render if dirty. Returns true if a frame was rendered.

Equivalent to tick_with_dt(events, 0) — animations are not advanced.

Source

pub fn tick_with_dt(&mut self, events: &[Event], dt_ms: u32) -> bool

Process events, advance animations by dt_ms milliseconds, and render if dirty. Returns true if a frame was rendered.

dt_ms: milliseconds elapsed since the last call. Pass 0 to skip animation advancement (useful for initial render or event-only ticks).

Source

pub fn mark_dirty(&mut self)

Force a full repaint on the next tick() call.
