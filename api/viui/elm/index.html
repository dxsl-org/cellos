Elm architecture — ViApp, Element<Msg>, free-function builders, macros.

iced-compatible API shape: ViApp::view() returns Element<Msg>. Free functions: text(), button(), column(), row(), checkbox(), scrollable(), space(), image(). Macros: column![], row![] for inline child lists.
