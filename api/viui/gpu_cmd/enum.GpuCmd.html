DrawImage

Blit raw BGRA pixels. Destination may extend beyond current clip — executor applies clipping during playback.

Fields

§dest: Rect
§pixels: Vec<u8>
§src_stride: u32
§

DrawTextShort

Zero-alloc path for text ≤ 127 bytes (covers all typical single-line UI strings). Bytes are always valid UTF-8 — written from &str in GpuCanvas::draw_text.
