pub struct RenderCtx<'a> {
    pub canvas: &'a mut dyn ViCanvas,
    pub font: &'a mut FontContext,
    pub theme: &'a dyn ViTheme,
}
Expand description

Combined draw surface + font state + active theme for one paint pass.

Passed by mutable reference through the entire widget tree during paint. Containers forward cx directly to children — no intermediate clone or reborrow.

Fields§

§canvas: &'a mut dyn ViCanvas

The pixel drawing surface for this frame.

§font: &'a mut FontContext

Scalable glyph state. atlas may be None (bitmap fallback).

§theme: &'a dyn ViTheme

Active design token set. Widgets read colors and spacing from here rather than hardcoding values.

pub fn char_width(&self) -> f32

Estimated character width at the current font size. Used for layout.

Returns an approximation — exact metrics require a full layout pass with the atlas. Widgets that need tight text layout should call self.font.atlas.as_mut().map(|a| a.advance(ch, px)) directly.
