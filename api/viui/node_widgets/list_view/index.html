ListView — scrollable list driven by Signal<Vec<String>>.

§Scroll behaviour

Mouse-wheel and touch-scroll update scroll_offset. A 4px scrollbar thumb is drawn when content exceeds the visible height. Touch fling inertia decays the scroll velocity at 0.85× per event until it falls below 0.5 px/event.

§Selection

Mouse-click or touch-begin selects the item under the pointer and fires on_select(idx). selected is a readable/writable Signal<Option<usize>> so callers can also set the selection programmatically.

§Variable item heights

Call .item_heights(signal) to enable variable-height mode. In this mode, scroll math uses a prefix-sum array; item_at() does a binary search. When the signal is absent the fixed-height code path is unchanged.
