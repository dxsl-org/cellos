Multi-screen navigation for ViUI apps.

§Stack navigation

Use StackNavigator when screens form a linear history (push/pop). Wrap it as the root widget of your crate::app_runner::ViApp.

§Tab navigation

Use TabNavigator for top-level sections with instant switching. Pages are built lazily on first activation and kept alive until the navigator is dropped.
