Expand description

Stack navigator — displays one page at a time.

Call push() / pop() to navigate; the navigator swaps the active page. Slide animation is scaffolded (slide_t, tick_animation) but painting falls back to the current page only until canvas transform support lands.

Wrap StackNavigator as the root widget of your ViApp. Call push() and pop() directly on the navigator (the inner Router is also pub for key inspection via router.current_key()).
