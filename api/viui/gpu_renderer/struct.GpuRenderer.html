Struct GpuRenderer 

Source 
pub struct GpuRenderer<E: CommandExecutor> { /* private fields */ }
Expand description

Generic renderer that records draw commands and replays via E.

Use GpuRenderer<CpuExecutor> as a drop-in replacement for FramebufferRenderer with damage-rect optimization enabled. Swap E for a hardware GPU executor in G2+ without touching widget code.
