Expand description

One overlay widget pushed onto the layer stack.

Fields§

§widget: Box<dyn ViNode>

The widget drawn on top of the base UI.

§blocking: bool

When true this overlay intercepts ALL input — nothing below it receives events. Use for modal dialogs.

§dismiss_outside: bool

When true a click outside widget.bounds() automatically pops this overlay. Use for drop-down menus and non-modal panels.

§anchor_bounds: Option<Rect>

If Some, the widget’s layout origin is constrained relative to this anchor rect (e.g. position a drop-down below its trigger button). When None the widget lays out against the full screen constraints.
