Struct GpuCanvas 

Source 
pub struct GpuCanvas<'buf> { /* private fields */ }
Expand description

Recording canvas: captures draw calls as GpuCmds.

'buf ties the canvas to the GpuCommandBuffer it records into; the buffer outlives the canvas and is consumed by the executor after painting.
