IPC send/recv round-trip latency benchmark.

Spawns a private echo peer (another instance of this binary in “ipc-echo” role) and measures the round-trip time of a 64-byte ping. This avoids any dependency on VFS TID, which can vary depending on the boot sequence. PDR target: < 50 µs per round-trip.
