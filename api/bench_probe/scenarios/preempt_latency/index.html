preempt_latency — RealTime wake-to-run latency under load.

The orchestrator sends a timestamped message to the RealTime probe; the probe (spawned via spawn_pinned at RealTime priority) wakes, reads the clock, and the orchestrator records t1 − t0. Sampled with N background load cells spinning, the delta approximates the scheduler’s preempt + IPC wake latency for a RealTime cell — the core RT-determinism figure.

⚠️ Runtime verification needs the bench cell embedded at /bin/bench (plan phase-05). Source compiles + type-checks standalone; numbers require a boot.
