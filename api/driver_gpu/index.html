GPU driver Cell — thin wrapper exposing sys_gpu_flush to sibling cells.

This cell has no IPC loop of its own; it provides library functions that the compositor cell (or any graphics-capable cell) calls directly, since all cells share the same address space.

The actual GPU hardware interaction happens in the kernel-side VirtIO GPU driver (kernel/src/task/drivers/virtio_gpu.rs) via the GpuFlush syscall.
