Hypha llm-gateway — the LLM network service cell.

Receives [LlmRequest] over IPC, does an HTTP chat-completion round-trip to an OpenAI-compatible endpoint, and replies [LlmReply]. Two transports (see transport.rs):

  • plaintext (USE_TLS = false, default): plain TCP via NetClient to a plain-HTTP mock (tools/hypha-mock-llm/mock_proxy.py --plain, port 8080). Easiest to test — no TLS variable.
  • TLS (USE_TLS = true): TLS 1.3 via the net service (port 8443).

Either way the cell holds no network capability — it only does IPC to the net cell. core spawns this cell and talks to it by tid (no registry in P1).
