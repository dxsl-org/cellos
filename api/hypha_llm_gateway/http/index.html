Minimal HTTP/1.0 + JSON helpers for P0/P2.

Hand-rolled on purpose (os-gaps G1 no HTTP lib, G4 no_std JSON): good enough to prove the LLM round-trip. Promote to ostd::http + serde-json-core once a second consumer appears or parsing needs to be robust against arbitrary provider responses.
