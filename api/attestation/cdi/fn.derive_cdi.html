pub fn derive_cdi(prev: &[u8; 32], layer_hash: &[u8; 32]) -> [u8; 32]
Expand description

Derive one CDI layer: HKDF-SHA256(salt = prev, ikm = layer_hash, info = "", L = 32). prev is the parent layer’s CDI (or the root secret for the first layer); layer_hash is that layer’s measurement (e.g. an ELF hash).
