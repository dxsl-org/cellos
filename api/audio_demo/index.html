Audio demo cell — proves the VirtIO sound path end-to-end.

Generates a 3-tone arpeggio (A4–C#5–E5) as a square wave and plays it through sys_audio_play. PCM format is fixed by the ABI: signed 16-bit LE, 2 channels (interleaved L/R), 44100 Hz. Square wave is used so no sin() (libm) is needed — purely integer generation.
