IPC server: receive Sign / Ecdh / GetPub requests; relay through mailbox; respond.

Wire layout (no unsafe ptr casts — all fields decoded manually):

SiloRequest (128 bytes): [0] opcode : u8 [1..31] _pad : [u8; 31] [32..127] data : [u8; 96]

MailboxPage (4096 bytes): [0..3] seq : u32 LE [4] cmd : u8 [5] resp : u8 [6..7] _pad : [u8; 2] [8..4095] data : [u8; 4088]

SiloResponse (128 bytes): [0] status : u8 [1] len : u8 [2..3] _pad : [u8; 2] [4..127] data : [u8; 124]
