Module vmm 

Source
Expand description

Low-level VMM syscall wrappers (220-226) for the silo service cell.

ARM64-only at runtime — guarded by the HypervisorCap manifest bit. RISC-V stubs return a NotSupported sentinel so the code compiles on all targets.

unsafe is required here for inline assembly (hardware syscall ABI). All other modules in this crate use #![forbid(unsafe_code)].
